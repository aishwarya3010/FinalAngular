About Us

Online room reservations are popular method for booking rooms. User can book rooms on a computer by using online security to protect their privacy and financial information and by using several online Client compare prices and facilities of different rooms.

Online accommodation is a web portal for providing details about the rooms available in users locality .User need to specify their requirement. The system will match the user requirement with the details in the database and after finding the match the required details is given to the user. It will also give information about facilities that are available nearby rooms. The system will have a Owner page from which owner can add details about the rooms. This will reduce the effort required by the new users and existing users in searching room and other useful things.
